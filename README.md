# LFMAlert

**LFMAlert** is a World of Warcraft addon that helps you never miss someone looking for more (LFM) in chat. Alerts appear as big messages in the middle of your screen, so you can see them instantly.

---

## Features

- Shows big, clear alerts when someone is looking for more.  
- Works in all chat channels, including guild, party, raid, say, yell, and whispers.  
- Easy to customize with your own keywords.  
- Quickly test how alerts look with a built-in command.  

---

## Usage

Use the **/lfm** command to manage keywords:

- `/lfm add <keyword>` – Add a keyword to watch for.  
- `/lfm remove <keyword>` – Remove a keyword.  
- `/lfm list` – See all your keywords.  
- `/lfm clear` – Remove all keywords.  
- `/lfm test` – Show a test alert on your screen.
